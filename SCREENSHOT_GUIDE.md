## 📋 需要截取的图片

### 1. 上传页面 (upload_page.png)

**步骤**：
1. 启动应用：`python recognition_app_v2.py`
2. 浏览器访问：http://127.0.0.1:5000
3. 等待页面完全加载
4. 截取完整页面（包含标题、上传区域）

**要点**：
- 展示科幻风格的UI设计
- 显示"选择图片"按钮
- 包含页面标题"角色识别 V2"

### 2. 识别结果页面 (detection_result.png)

**步骤**：
1. 在上传页面选择一张角色图片
2. 点击"开始识别"
3. 等待识别完成
4. 截取结果页面

**要点**：
- 左侧：图片上的绿色/黄色检测框
- 右侧：识别结果列表
- 显示角色名称和置信度
- 最好选择识别准确的图片

**推荐测试图片**：
- 使用 `战双人物图像_调整尺寸/` 中的图片
- 选择清晰、正面的角色图片
- 建议使用luxiya、lee或lifu的图片（样本多，准确率高）

### 3. 训练曲线 (training_curves.png)

**步骤**：
- 已自动复制到 `screenshots/` 文件夹
- 无需额外操作

### 4. 多角色识别 (multi_character.png) - 可选

**步骤**：
1. 找一张包含多个角色的图片
2. 上传并识别
3. 截取显示多个检测框的结果

## 🖼️ 截图方法

### Windows

**方法1：截图工具（推荐）**
```
Win + Shift + S
```
- 选择区域截图
- 自动复制到剪贴板
- 粘贴到画图或其他工具保存

**方法2：全屏截图**
```
Win + PrtScn
```
- 保存到 `图片/屏幕截图` 文件夹

**方法3：Snipping Tool**
- 搜索"截图工具"
- 选择区域截图
- 保存为PNG

### 浏览器截图（推荐）

**Chrome/Edge**：
1. 按 `F12` 打开开发者工具
2. 按 `Ctrl + Shift + P`
3. 输入 "screenshot"
4. 选择 "Capture full size screenshot"

## 📐 图片规格

### 尺寸建议
- **宽度**: 1280px - 1920px
- **高度**: 根据内容自适应
- **比例**: 16:9 或 16:10

### 文件要求
- **格式**: PNG（推荐）或JPG
- **质量**: 高清，文字清晰
- **大小**: 单张 < 2MB
- **命名**: 使用英文，小写加下划线

## ❓ 常见问题

### Q: 截图太大怎么办？

A: 使用图片压缩工具：
- TinyPNG: https://tinypng.com/
- 或使用画图工具调整大小

### Q: 识别效果不好怎么办？

A: 
- 换一张清晰的测试图片
- 使用样本多的角色（luxiya, lee, lifu）
- 确保图片中角色脸部清晰可见

### Q: 需要添加水印吗？

A: 不需要，保持原始截图即可
