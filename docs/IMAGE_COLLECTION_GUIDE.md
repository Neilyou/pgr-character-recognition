## 📋 图片质量标准

### ✅ 合格图片

1. **清晰度**
   - 分辨率 ≥ 512x512
   - 无明显模糊
   - 无严重压缩痕迹

2. **角色可见性**
   - 角色占图片面积 ≥ 30%
   - 角色主要特征清晰可见
   - 面部、服装、武器等特征完整

3. **背景**
   - 背景不过于复杂
   - 角色与背景有明显区分
   - 无过多干扰元素

4. **角度和姿态**
   - 多样化的角度（正面、侧面、背面）
   - 不同姿态（站立、战斗、特写）
   - 避免过于相似的图片

5. **格式**
   - JPG、PNG、WEBP
   - 文件大小 < 10MB

### ❌ 不合格图片

1. **质量问题**
   - 分辨率过低（<256x256）
   - 严重模糊或失真
   - 过度压缩

2. **内容问题**
   - 角色被严重遮挡（>50%）
   - 多个角色混杂难以区分
   - 包含大量文字水印

3. **其他问题**
   - 表情包、梗图
   - 过度二次创作（风格差异大）
   - 色情、暴力等不适内容

## 💡 收集技巧

### 1. 分批收集
- 先收集优先级最高的5个角色（qu, shenwei, wanshi, dubian, qishi）
- 每个角色收集到30张后再继续下一个
- 避免一次性收集过多导致疲劳

### 2. 多样化
- 不同角度（正面、侧面、背面）
- 不同姿态（站立、战斗、技能释放）
- 不同场景（战斗、休闲、特写）
- 不同服装（如果有多个皮肤）

### 3. 质量优先
- 宁缺毋滥
- 一张高质量图片 > 三张低质量图片
- 定期检查已收集图片的质量

### 4. 记录来源
- 记录图片来源便于后续补充
- 标注版权信息
- 保存原始链接
